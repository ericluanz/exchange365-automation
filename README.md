# Exchange Online 365 - Scripts & Automação

Scripts e automações para gerenciamento do Exchange Online (Microsoft 365) utilizados no suporte e administração no meu dia a dia.

---

## 🔧 Exemplos

### 📧 Mailbox
- Criar caixa compartilhada
- Converter usuário em Shared Mailbox
- Aplicar quotas
- Ativar Archive Mailbox

### 🔐 Permissions
- Adicionar Full Access
- Send As / Send on Behalf
- Auditoria de permissões

### 👥 Groups
- Criar Distribution List
- Adicionar membros em massa
- Exportar membros

### 📊 Reports
- Relatório de caixas inativas
- Licenças atribuídas
- Uso de armazenamento

### 🛡 Security
- Auditoria de logins
- Regras de encaminhamento externo
- Verificação de Inbox Rules suspeitas

---

## ⚙️ Requisitos

- PowerShell 7+
- Exchange Online Management Module
- Permissões administrativas no Microsoft 365

Instalar módulo:

```powershell
Install-Module ExchangeOnlineManagement
